## 方式一：Web 界面操作 🌐

**推荐指数：** ⭐⭐⭐⭐⭐  
**适合人群：** 所有用户

### 启动方式

```batch
# 双击启动
D:\RAG_Knowledge_Base\web\启动 Web 界面.bat
```

### 访问地址

```
http://localhost:5000
```

### 核心功能

#### 1. 文档上传

- **拖拽上传**：拖拽文件到上传区域
- **点击上传**：点击选择文件
- **支持格式**：PDF、DOCX、TXT、Excel、PPTX

#### 2. 索引管理

- **刷新索引**：点击"刷新索引"按钮
- **查看状态**：显示文档数量和文本块数
- **增量更新**：只处理新增/修改的文档

#### 3. 智能问答

- **输入问题**：在查询框输入问题
- **获取答案**：AI 生成答案并标注来源
- **查看引用**：点击来源查看原文档

#### 4. 文件管理

- **查看文件列表**：显示所有已索引文档
- **删除文件**：删除不需要的文档
- **OCR 处理**：对图片 PDF 进行 OCR 识别

## 方式二：命令行操作 💻

**推荐指数：** ⭐⭐⭐⭐  
**适合人群：** 技术人员、批量操作

### 启动交互式工具

```batch
# 方式 1：双击启动
D:\RAG_Knowledge_Base\tools\RAG 知识库管理工具.bat

# 方式 2：命令行启动
cd D:\RAG_Knowledge_Base\tools
python rag_interactive.py
```

### 常用命令

#### 查询知识库

```batch
# 查询问题
python core\core.py query "问题内容"

# 带参数查询
python core\core.py query "问题内容" --top_k 2 --auto_refresh false
```

#### 管理索引

```batch
# 刷新索引
python core\core.py refresh

# 强制重建索引
python core\core.py refresh --force

# 查看索引状态
python core\core.py status
```

#### OCR 处理

```batch
# 处理单个 PDF
python tools\ocr_simple.py "文件.pdf"

# 批量处理目录
python tools\ocr_processor.py --batch "目录路径"
```

## 方式三：Python API 调用 🐍

**推荐指数：** ⭐⭐⭐⭐  
**适合人群：** 开发人员

### 导入模块

```python
from core.core import RAGKnowledgeBase

# 创建知识库实例
rag = RAGKnowledgeBase(
    documents_path=r"D:\RAG_Knowledge_Base\documents",
    chroma_db_path=r"D:\RAG_Knowledge_Base\documents\.rag\chroma_db"
)
```

### 查询知识库

```python
# 基本查询
result = rag.query("问题内容")
print(result["answer"])
print(result["sources"])

# 带参数查询
result = rag.query(
    "问题内容",
    top_k=2,
    auto_refresh=False
)
```

### 管理索引

```python
# 刷新索引
rag.refresh_index()

# 查看状态
status = rag.get_status()
print(f"文档数：{status['doc_count']}")
print(f"文本块数：{status['block_count']}")
```

## 方式四：AI 助手集成 🤖

**推荐指数：** ⭐⭐⭐⭐⭐  
**适合人群：** 所有用户（最便捷）

### 通过 CoPaw 调用

```python
# 在 CoPaw 工作区中
from active_skills.rag_knowledge.rag_skill import rag_query

# 查询知识库
result = rag_query("问题内容")
print(result)
```

### 通过 OpenClaw 调用（自然语言）

**配置 Skill 后，可直接对话：**

```
用户：查询一下知识库中关于电费的信息
AI:  [自动调用 RAG 知识库技能]
     根据知识库文档，电费相关信息如下：
     - 2026 年 2 月电费金额：[数值] 元
     - 用电量：[数值] 千瓦时
     - 来源：[文档名称]
```

### Skill 配置

**OpenClaw 配置示例：**

```json
{
  "skills": [
    {
      "name": "rag_knowledge",
      "type": "python",
      "path": "<INSTALL_DIR>\\skills\\rag_knowledge",
      "description": "RAG 知识库查询技能",
      "triggers": ["查询知识库", "搜索文档", "查找资料"]
    }
  ]
}
```
