## 🔍 查询操作

### Web 界面查询

1. 打开 http://localhost:5000
2. 在查询框输入问题
3. 点击"查询"按钮
4. 查看答案和来源

### 命令行查询

```batch
# 基本查询
python D:\RAG_Knowledge_Base\core\core.py query "问题内容"

# 带参数查询
python D:\RAG_Knowledge_Base\core\core.py query "问题内容" --top_k 2 --auto_refresh false

# 交互式查询
python D:\RAG_Knowledge_Base\tools\rag_interactive.py
```

### Python API 查询

```python
from core.core import RAGKnowledgeBase

rag = RAGKnowledgeBase()

# 查询
result = rag.query("问题内容")
print(result["answer"])

# 带参数查询
result = rag.query("问题内容", top_k=2, auto_refresh=False)
```

### AI 助手查询

```python
# CoPaw
from active_skills.rag_knowledge.rag_skill import rag_query
result = rag_query("问题内容")
```

## 📊 常用参数

### 查询参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `top_k` | 2 | 检索文档数量 |
| `auto_refresh` | false | 自动刷新索引 |
| `similarity_threshold` | 0.3 | 相似度阈值 |

### 示例

```bash
# 检索 3 个文档
python core.py query "问题" --top_k 3

# 关闭自动刷新
python core.py query "问题" --auto_refresh false

# 组合参数
python core.py query "问题" --top_k 2 --auto_refresh false
```

## 🐍 Python 代码片段

### 基本查询

```python
from core.core import RAGKnowledgeBase

rag = RAGKnowledgeBase()
result = rag.query("问题内容")

print("答案:", result["answer"])
print("来源:", result["sources"])
```

### 批量查询

```python
questions = ["问题 1", "问题 2", "问题 3"]

for q in questions:
    result = rag.query(q)
    print(f"Q: {q}")
    print(f"A: {result['answer']}\n")
```

### 索引管理

```python
# 刷新索引
rag.refresh_index()

# 查看状态
status = rag.get_status()
print(status)

# 添加文档
rag.add_document("新文档.pdf")

# 删除文档
rag.delete_document("旧文档.pdf")
```
