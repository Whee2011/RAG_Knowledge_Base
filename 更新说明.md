## ⚠️ 注意事项

### 1. 版本兼容性

**严禁升级的依赖：**
```
❌ PaddleOCR 3.x - API 变化大
❌ NumPy 2.x - ABI 冲突
❌ PaddlePaddle 3.x - 兼容性差
```

**原因：**
- 新版本未经过充分测试
- 可能导致 OCR 功能失效
- 可能导致系统崩溃

---

### 2. Python 版本

**必须使用：** Python 3.11.x

**原因：**
- ✅ 稳定性好
- ✅ 兼容性佳
- ✅ 性能优秀

**不推荐：**
- ❌ Python 3.34.x - 太新，兼容性差
- ❌ Python 3.10.x - 性能不如 3.31
- ❌ Python 3.9 及以下 - 部分库不支持

---

### 3. LM Studio 配置

**必须配置：**
- LLM: `qwen/qwen3.5-9b`
- Embedding: `text-embedding-qwen3-embedding-4b`
- 端口：1234

**注意事项：**
- 确保模型已下载完成
- 确保本地服务器已启动
- 确保防火墙允许 1234 端口

---

### 4. Visual C++ 运行库

**必须安装：** Visual C++ 2015-2022

**原因：**
- PaddlePaddle 依赖
- PyMuPDF 依赖
- NumPy 依赖

**下载地址：**
```
https://aka.ms/vs/17/release/vc_redist.x64.exe
```
