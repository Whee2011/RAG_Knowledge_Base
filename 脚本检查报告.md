## 📦 脚本功能说明

### 1. install.bat - 主安装向导

**功能：**
- 选择安装路径（D 盘/C 盘/E 盘/自定义）
- 系统环境检测
- Python 环境安装
- VC++ 运行库安装
- 依赖包安装
- 文件复制
- 快捷方式创建

**流程：**
```
选择路径 → 环境检测 → 安装 Python → 安装 VC++ 
→ 安装依赖 → 复制文件 → 创建快捷方式 → 完成
```

---

### 2. start_all.bat - 启动所有服务

**功能：**
- 检查 LM Studio 状态
- 启动 RAG Web 服务
- 启动 OCR 自动处理服务（可选）
- 验证服务状态
- 自动打开浏览器

**启动顺序：**
```
1. LM Studio (端口 1234)
2. RAG Web 服务 (端口 5000)
3. OCR 自动处理服务 (可选)
```

---

### 3. stop_all.bat - 停止所有服务

**功能：**
- 停止 RAG Web 服务
- 停止 OCR 自动处理服务
- 验证服务已停止
- 保留 LM Studio（手动关闭）

---

### 4. status.bat - 服务状态检测

**功能：**
- LM Studio 状态
- RAG Web 服务状态
- OCR 服务状态
- Python 环境状态
- 文件系统检测
- 依赖包检测
- 快速操作菜单

---

### 5. check_environment.bat - 环境检测

**检测项目：**
1. 系统信息（Windows 版本、计算机名、用户）
2. 硬件信息（CPU、内存）
3. 磁盘空间
4. Python 环境
5. Visual C++ 运行库 ⭐ 新增
6. LM Studio
7. Python 依赖包
8. 网络连通性
9. 端口占用
10. 检测总结

**输出：** 生成 `environment_report.txt` 报告文件

---

### 6. fix_vcredist.bat - VC++ 修复工具

**功能：**
- 检测缺失的 DLL 文件
- 提供 3 种安装方式：
  1. 本地安装包（推荐）
  2. 网络下载安装
  3. 手动下载安装
- 安装后验证
- 提示重启系统

**支持的 DLL：**
- `vcruntime140.dll`
- `msvcp140.dll`
- `vcruntime140_1.dll`（可选）

---

### 7. download_packages.bat - 离线包下载

**下载内容：**
1. Visual C++ 运行库（约 25MB）
2. Python 3.10 嵌入式（约 30MB）
3. Python 依赖包（约 200-500MB）

**输出目录：**
```
packages/
├── vcredist_x64.exe

├── requirements.txt
└── requirements/
    ├── chromadb-*.whl
    ├── flask-*.whl
    ├── pymupdf-*.whl
    └── ...
```

## 📝 使用说明

### 分发安装包结构

```
RAG_Knowledge_Base_Installer_v2.0/
│
├── install.bat              ⭐ 主安装脚本
├── start_all.bat            启动脚本
├── stop_all.bat             停止脚本
├── status.bat               状态检测
├── check_environment.bat    环境检测
├── fix_vcredist.bat         VC++ 修复
├── download_packages.bat    离线包下载
│
├── packages/                ⭐ 离线包目录
│   ├── vcredist_x64.exe
│   ├── python_embedded.zip
│   └── requirements/
│       └── *.whl
│
├── core/                    核心代码
├── web/                     Web 界面
├── tools/                   工具脚本
├── docs/                    文档
│
└── config.yaml.template     配置模板
```

### 安装步骤

**方法 1：完整离线安装**
```
1. 下载完整安装包（含 packages 目录）
2. 右键 install.bat → 以管理员身份运行
3. 选择安装路径
4. 等待安装完成
5. 启动服务
```

**方法 2：在线安装**
```
1. 下载精简安装包（不含 packages）
2. 运行 download_packages.bat 下载依赖
3. 右键 install.bat → 以管理员身份运行
4. 选择安装路径
5. 等待安装完成
```

3. **需要管理员权限** - 安装和 VC++ 修复需要
4. **首次启动较慢** - 需下载 OCR 模型（约 200MB）

## 📞 故障排查

### 常见问题

**Q1: 安装时提示"需要管理员权限"**
```
解决：右键点击 install.bat，选择"以管理员身份运行"
```

**Q2: VC++ 安装失败**
```
解决：
1. 运行 fix_vcredist.bat
2. 选择"手动安装"
3. 从官网下载安装包
4. 手动运行安装
```

**Q3: 依赖包下载失败**
```
解决：
1. 检查网络连接
2. 使用 download_packages.bat 预下载
3. 使用离线包安装
```

**Q4: 服务启动失败**
```
解决：
1. 运行 status.bat 查看状态
2. 检查端口是否被占用
3. 查看日志文件
```
