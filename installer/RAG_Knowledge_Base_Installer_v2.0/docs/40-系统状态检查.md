## 📊 详细检查

### 1. 服务状态检查

#### LM Studio 服务

```batch
# 检查端口
netstat -ano | findstr ":1234"

# 检查进程
tasklist | findstr "lm-studio"
```

**正常状态：**
- ✅ 端口 1234 监听中
- ✅ 进程运行中

**异常处理：**
```
❌ 端口未监听 → 启动 LM Studio
❌ 进程不存在 → 启动 LM Studio 应用
```

#### RAG Web 服务

```batch
# 检查端口
netstat -ano | findstr ":5000"

# 检查进程
tasklist | findstr "web_app.py"
```

**正常状态：**
- ✅ 端口 5000 监听中
- ✅ 进程运行中

**异常处理：**
```batch
# 启动 Web 服务
D:\RAG_Knowledge_Base\web\启动 Web 界面.bat
```

#### OCR 服务

```batch
# 检查 OCR 进程
tasklist /FI "WINDOWTITLE eq RAG OCR*"
```

**正常状态：**
- ✅ 进程运行中（如启用）
- ℹ️  未运行（可选服务）

## 🚨 常见异常状态

### 1. LM Studio 未运行

**症状：**
```
❌ LM Studio: 未运行
❌ 端口 1234: 未监听
```

**解决：**
```
1. 启动 LM Studio 应用
2. 加载 qwen/qwen3.5-9b 模型
3. 启动本地服务器（端口 1234）
```

### 2. Web 服务未运行

**症状：**
```
❌ Web 服务：未运行
❌ 端口 5000: 未监听
```

**解决：**
```batch
D:\RAG_Knowledge_Base\web\启动 Web 界面.bat
```

### 3. 索引损坏

**症状：**
```
❌ ChromaDB: 未初始化
❌ 查询失败：Collection 不存在
```

**解决：**
```batch
# 重建索引
python D:\RAG_Knowledge_Base\core\core.py refresh --force
```

### 4. 依赖包缺失

**症状：**
```
❌ chromadb: 未安装
❌ ImportError: No module named 'chromadb'
```

**解决：**
```batch
# 重新安装
pip install chromadb flask pymupdf rapidocr
```

## 📈 性能监控

### 查询性能

```python
import time
from core.core import RAGKnowledgeBase

rag = RAGKnowledgeBase()

start = time.time()
result = rag.query("测试问题")
end = time.time()

print(f"查询耗时：{end - start:.2f}秒")
print(f"Prompt Token: {result['stats']['prompt_tokens']}")
print(f"Completion Token: {result['stats']['completion_tokens']}")
```

**正常范围：**
- 查询耗时：2-10 秒
- Prompt Token: 1000-5000
- Completion Token: 100-500
