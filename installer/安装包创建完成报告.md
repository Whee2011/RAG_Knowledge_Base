## 🎯 安装包特点

### ✅ 优势

1. **完全离线安装** - 无需联网即可完成安装
2. **一键安装** - 运行 `install.bat` 自动完成所有步骤
3. **VC++ 自动安装** - 自动检测并安装 Visual C++ 运行库
4. **环境检测** - 安装前全面检测系统环境
5. **完整文档** - 24 个文档覆盖所有使用场景
6. **Web 界面** - 提供直观的浏览器操作界面
7. **OCR 功能** - 内置 PaddleOCR 支持图片 PDF 识别
8. **版本锁定** - 关键依赖版本锁定，避免兼容性问题

### ⚠️ 注意事项

1. **需要管理员权限** - 安装和 VC++ 修复需要管理员权限
2. **仅支持 64 位系统** - Windows 10/11 64 位
3. **首次启动较慢** - 首次运行需下载 OCR 模型（约 200MB）
4. **磁盘空间** - 建议至少 50GB 可用空间

## 📞 技术支持

### 常见问题

**Q1: 安装时提示"需要管理员权限"**
```
解决：右键点击 install.bat，选择"以管理员身份运行"
```

**Q2: VC++ 安装失败**
```
解决：
1. 运行 fix_vcredist.bat
2. 选择"手动安装"
3. 从官网下载安装包手动安装
```

**Q3: 依赖包验证失败**
```
解决：
1. 检查 packages/requirements 目录是否存在
2. 确认有 81 个 .whl 文件
3. 重新运行 download_packages.bat
```

**Q4: LM Studio 未检测到**
```
解决：
1. 先安装 LM Studio
2. 下载地址：https://lmstudio.ai/
3. 安装完成后再运行 RAG 安装程序
```
