4. **配置 LM Studio**
   - 下载并安装 LM Studio
   - 下载 LLM 模型：`qwen/qwen3.5-9b`
   - 下载 Embedding 模型：`text-embedding-qwen3-embedding-4b`
   - 启动本地服务器（端口 1234）

5. **开始使用**
   - Web 界面：http://localhost:5000
   - 交互式工具：`tools\RAG 知识库管理工具.bat`
