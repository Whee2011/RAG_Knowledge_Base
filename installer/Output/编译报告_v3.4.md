## 🚀 使用方式

### 安装

```batch
# 双击运行
RAG-Local_v3.4_Setup.exe

# 或命令行安装（静默）
RAG-Local_v3.4_Setup.exe /VERYSILENT
```

### 测试混合检索

```python
from core.core import KnowledgeBase

kb = KnowledgeBase()

# 简单问题（向量检索）
result = kb.query("2 月电费是多少？")
# 日志：[INFO] 简单问题，使用向量检索

# 复杂问题（混合检索）
result = kb.query("为什么 A 公司比 B 公司利润高？")
# 日志：[INFO] 复杂问题，使用混合检索
```
