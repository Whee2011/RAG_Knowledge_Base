## 🚀 安装流程

在目标电脑上的安装步骤：

1. **复制安装包**
   ```
   将整个 RAG_Knowledge_Base_Installer_v2.0 文件夹复制到目标电脑
   ```

2. **运行安装脚本**
   ```
   右键点击 install.bat → 选择"以管理员身份运行"
   ```

3. **等待安装完成**
   - 安装 Python 3.11.9 嵌入式版本
   - 安装 VC++ 运行库
   - 安装 Python 依赖包
   - 复制核心代码

4. **安装 LM Studio**
   - 下载：https://lmstudio.ai/
   - 加载 LLM 模型（qwen3.5-9b）
   - 加载 Embedding 模型

5. **启动系统**
   ```
   D:\RAG_Knowledge_Base\web\启动 Web 界面.bat
   ```

---

## ⚠️ 注意事项

1. **Python 版本必须匹配**
   - 安装包使用 Python 3.11.9 嵌入式版本
   - 所有 whl 包都是 cp311 或 cp310-abi3 格式
   - 不要混用其他 Python 版本

2. **LM Studio 和模型需要单独提供**
   - LM Studio 主程序（需从官网下载）
   - LLM 模型文件（通常较大，需单独传输）
   - Embedding 模型文件

3. **安装路径建议**
   - 推荐：D:\RAG_Knowledge_Base
   - 避免：包含中文或空格的路径
